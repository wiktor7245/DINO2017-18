Początek :)

git config --global user.email "wiktor7245@o2.pl" - Konfiguracja własnego adresu e-mail.

git commit -a -m "wiadomość" - Zapis wszystkich zmian w repozytorium z wiadomością podaną w cudzysłowiu po opcji -m.

git diff --staged - Pokazuje zmiany między pomiędzy commitami

git checkout -b "branch" - Tworzy gałąź o nazwie branch i się na nią przełącza.

git log --graph --all - Wyświetla wszystkie wykresy zmian w repozytorium.

git branch -d "branch" - Usuwa gałąź o nazwie branch.

git remote add "remote" "https://github.com/wiktor7245/SomeRepo" - Dodaje zdalne repozytorium o nazwie remote do repozytorium na stronie https://github.com/wiktor7245/SomeRepo.

git fetch "remote" "branch" - Pobiera informacje ze zdalengo repozytorium o nazwie remote do gałęzi o nazwie branch.
